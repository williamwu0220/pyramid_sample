This project allows you to upload multiple files at the same time.
Before give a test for this project, you should make a new folder named 'posimage'
and put it in the static folder of the project, or you will get an exception.

How to setup the project successfully on Windows:
Before you try to setup this project (python setup.py develop) on Windows,
visit this link: http://www.lfd.uci.edu/~gohlke/pythonlibs/#pil,
and download the related Pillow package and install it into your Windows system.
After you installed it successfully, copy PIL\ and Pillow-2.3.0-py3.3.egg-info\ in C:\Python33\Lib\site-packages\
to YOUR_VIRTUALENV_FOLDER\Lib\site-packages\.
